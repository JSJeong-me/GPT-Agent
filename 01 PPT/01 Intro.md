# 실습 Note

## ChatGPT와 NotebookLM을 활용한 PPT 작성 실습

### 실습 시간

1시간

### 실습 주제

**문서 기반으로 설득형 PPT 초안 만들기**

---

## 1. 실습 목표

본 실습의 목표는 ChatGPT와 NotebookLM을 함께 활용하여 **근거 있는 PPT 초안**을 빠르게 설계하는 것이다.

교육생은 NotebookLM을 활용해 문서에서 핵심 내용과 근거를 추출하고, ChatGPT를 활용해 발표 목적에 맞는 PPT 목차, 결론형 제목, 슬라이드 구성, 발표자 노트, 예상 Q&A를 작성한다.

---

## 2. 실습 핵심 개념

### 1) ChatGPT의 역할

ChatGPT는 PPT의 **구조와 메시지 설계**에 강점이 있다.

주요 활용 예시는 다음과 같다.

* 발표 목적 정리
* 청중 분석
* PPT 목차 작성
* 결론형 슬라이드 제목 작성
* 슬라이드별 핵심 메시지 작성
* 발표자 노트 작성
* 예상 질문과 답변 작성
* 텍스트를 표, 다이어그램, 비교 구조로 변환

---

### 2) NotebookLM의 역할

NotebookLM은 업로드한 문서나 자료를 기반으로 답변을 생성하는 데 강점이 있다.

주요 활용 예시는 다음과 같다.

* 문서 요약
* 핵심 주장 추출
* 근거 문장 확인
* 문서 기반 질의응답
* FAQ 생성
* 브리핑 문서 작성
* 발표자료에 넣을 신뢰성 있는 내용 정리

---

### 3) 두 도구의 역할 구분

| 구분    | ChatGPT                 | NotebookLM               |
| ----- | ----------------------- | ------------------------ |
| 핵심 역할 | 발표자료 구조 설계              | 문서 기반 근거 추출              |
| 강점    | 목차, 메시지, 스토리라인, 표현 개선   | 원문 기반 요약, 질의응답, 근거 확인    |
| 활용 시점 | PPT 기획 및 슬라이드 작성        | 자료 분석 및 내용 검증            |
| 결과물   | PPT 목차, 슬라이드 초안, 발표자 노트 | 문서 요약, 인용 근거, FAQ, 핵심 쟁점 |

---

## 3. 실습 준비물

실습 전 다음 자료를 준비한다.

* 발표 주제 1개
* 참고 문서 1~3개

  * 보고서
  * 제안서
  * 회의록
  * 정책 문서
  * 제품 설명서
  * 기사 또는 PDF 자료
* ChatGPT 접속 환경
* NotebookLM 접속 환경
* PPT 작성 도구

  * PowerPoint
  * Google Slides
  * Gamma
  * 미리캔버스
  * 기타 프레젠테이션 도구

---

## 4. 실습 산출물

실습이 끝나면 다음 결과물을 만든다.

* 8장 내외 PPT 목차
* 슬라이드별 결론형 제목
* 슬라이드별 핵심 메시지
* 근거 자료 요약
* 발표자 노트 초안
* 예상 질문과 답변 5개
* PPT 작성용 최종 프롬프트

# 5. 실습 진행 순서

---

## Step 1. 발표 주제와 청중 정하기

### 시간: 0 ~ 5분

먼저 작성할 PPT의 주제와 청중을 정한다.

### 예시 주제

* 생성형 AI를 활용한 업무혁신 제안
* 사내 RAG 시스템 도입 제안
* AI 기반 고객문의 자동 분류 시스템
* 영업 보고 자동화 방안
* 사내 문서 검색 효율화 방안

### 작성 항목

| 항목         | 작성 내용 |
| ---------- | ----- |
| 발표 주제      |       |
| 발표 대상      |       |
| 발표 목적      |       |
| 발표 후 기대 행동 |       |

### 예시

| 항목         | 작성 내용                  |
| ---------- | ---------------------- |
| 발표 주제      | 생성형 AI를 활용한 업무혁신 제안    |
| 발표 대상      | 부서장 및 실무 리더            |
| 발표 목적      | AI 도구 도입 필요성과 활용 방안 제안 |
| 발표 후 기대 행동 | 파일럿 프로젝트 승인            |

---

## Step 2. NotebookLM에 자료 업로드하기

### 시간: 5 ~ 12분

NotebookLM에 참고 문서를 업로드한다.

### 작업 순서

1. NotebookLM에서 새 Notebook을 만든다.
2. 참고 문서 또는 PDF를 업로드한다.
3. 문서가 정상적으로 인식되었는지 확인한다.
4. 문서의 주요 내용을 먼저 요약시킨다.

### NotebookLM 프롬프트 예시

```text
업로드한 문서를 바탕으로 핵심 내용을 5개 항목으로 요약해 주세요.
각 항목은 발표자료에 사용할 수 있도록 간결한 문장으로 작성해 주세요.
```

```text
이 문서에서 발표자료에 넣을 수 있는 핵심 주장, 근거, 수치, 사례를 표로 정리해 주세요.
열은 핵심 주장, 근거 내용, 출처 위치, PPT 활용 방식으로 구성해 주세요.
```

---

## Step 3. NotebookLM으로 근거 자료 추출하기

### 시간: 12 ~ 25분

PPT에 사용할 근거를 NotebookLM에서 추출한다.

### 확인할 내용

* 문서의 핵심 주장
* 문제점 또는 배경
* 주요 수치
* 사례
* 기대 효과
* 리스크 또는 한계
* 발표에서 강조할 문장

### NotebookLM 프롬프트 예시

```text
이 문서의 내용을 바탕으로 발표자료에 사용할 핵심 메시지 5개를 작성해 주세요.
각 메시지마다 근거가 되는 문서 내용을 함께 정리해 주세요.
```

```text
이 자료를 임원 보고용 PPT에 활용하려고 합니다.
의사결정자가 관심을 가질 만한 비용, 효과, 리스크, 실행 가능성 관점의 내용을 정리해 주세요.
```

```text
이 문서를 바탕으로 발표자가 받을 수 있는 예상 질문 5개와 답변 방향을 작성해 주세요.
```

### 산출물

* 핵심 메시지 5개
* 근거 자료 5개
* 예상 질문 5개

---

## Step 4. ChatGPT로 PPT 스토리라인 만들기

### 시간: 25 ~ 40분

NotebookLM에서 정리한 내용을 ChatGPT에 입력하여 PPT 목차와 스토리라인을 만든다.

### ChatGPT 프롬프트 예시

```text
당신은 전략 컨설턴트입니다.

다음 내용을 바탕으로 8장짜리 발표자료 목차를 작성해 주세요.

[발표 주제]
생성형 AI를 활용한 업무혁신 제안

[발표 대상]
부서장 및 실무 리더

[발표 목적]
AI 도구 도입 필요성과 파일럿 프로젝트 추진 필요성을 설득

[NotebookLM에서 정리한 핵심 내용]
1. ...
2. ...
3. ...

[작성 조건]
- 슬라이드 제목은 결론형 문장으로 작성
- 각 슬라이드마다 핵심 메시지 1개 포함
- 각 슬라이드마다 추천 시각자료 포함
- 마지막에는 실행 계획과 기대 효과 포함

[출력 형식]
슬라이드 번호 / 결론형 제목 / 핵심 메시지 / 주요 내용 / 추천 시각자료
```

### 권장 PPT 구조

| 슬라이드 | 역할                        |
| ---- | ------------------------- |
| 1    | 제목 및 핵심 제안                |
| 2    | 현재 업무 문제                  |
| 3    | 왜 AI 도입이 필요한가             |
| 4    | ChatGPT와 NotebookLM 활용 방식 |
| 5    | 적용 가능한 업무 영역              |
| 6    | 기대 효과                     |
| 7    | 추진 일정                     |
| 8    | 요청 사항 및 다음 단계             |

---

## Step 5. ChatGPT로 슬라이드별 초안 작성하기

### 시간: 40 ~ 50분

스토리라인을 바탕으로 슬라이드별 내용을 구체화한다.

### ChatGPT 프롬프트 예시

```text
위에서 작성한 8장 PPT 목차를 바탕으로 각 슬라이드의 상세 내용을 작성해 주세요.

각 슬라이드는 다음 형식으로 작성해 주세요.

1. 슬라이드 제목
2. 핵심 메시지
3. 본문 bullet 3개
4. 추천 시각자료
5. 발표자 노트 3문장
```

### 슬라이드 작성 기준

* 한 슬라이드에는 하나의 메시지만 담는다.
* 제목은 주제형이 아니라 결론형으로 작성한다.
* 본문은 3개 bullet 이내로 정리한다.
* 근거는 NotebookLM에서 확인한 내용만 사용한다.
* 자세한 설명은 발표자 노트로 이동한다.

---

## Step 6. PPT 품질 검토하기

### 시간: 50 ~ 57분

작성된 PPT 초안을 검토한다.

### 검토 기준

| 검토 항목   | 확인 질문                         |
| ------- | ----------------------------- |
| 목적 적합성  | 발표 목적이 분명한가?                  |
| 청중 적합성  | 청중이 관심 가질 내용인가?               |
| 메시지 명확성 | 제목만 읽어도 핵심이 보이는가?             |
| 근거 신뢰성  | 문서 기반 근거가 있는가?                |
| 구성 흐름   | 문제 → 해결 → 실행 → 효과 흐름이 자연스러운가? |
| 실행 가능성  | 실제 업무에 적용 가능한가?               |
| 위험 관리   | 검토와 승인 절차가 포함되어 있는가?          |

### ChatGPT 검토 프롬프트

```text
다음 PPT 초안을 검토해 주세요.

검토 기준은 다음과 같습니다.
1. 발표 목적 적합성
2. 청중 적합성
3. 메시지 명확성
4. 근거 신뢰성
5. 슬라이드 흐름
6. 실행 가능성
7. 예상 질문 대응력

각 기준별로 보완할 점을 작성하고, 수정된 슬라이드 제목을 제안해 주세요.
```

# 8. 실습 핵심 정리

ChatGPT와 NotebookLM을 함께 사용하면 PPT 작성 과정이 다음과 같이 바뀐다.

| 기존 방식            | AI 활용 방식                 |
| ---------------- | ------------------------ |
| 사람이 문서를 직접 읽고 요약 | NotebookLM이 문서 기반 요약 제공  |
| 사람이 목차를 처음부터 작성  | ChatGPT가 목적별 목차 초안 생성    |
| 제목이 단순 주제 중심     | 결론형 메시지 제목으로 개선          |
| 근거 확인이 어려움       | NotebookLM에서 문서 기반 근거 확인 |
| 발표 준비가 별도 작업     | ChatGPT가 발표자 노트와 Q&A 생성  |

---

## 핵심 문장

> NotebookLM은 근거를 찾고, ChatGPT는 발표 구조를 만든다.
> 좋은 PPT는 AI가 자동으로 만들어 주는 것이 아니라, 사람이 목적과 청중을 정하고 AI가 초안을 빠르게 확장하도록 설계할 때 완성된다.
